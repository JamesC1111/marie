# Marie Harding Counselling Service

Astro + TypeScript + Tailwind website for Marie Harding Counselling Service in Kanturk, Co. Cork.

## Start

- `npm install`
- `npm run dev`

Open [http://localhost:4321](http://localhost:4321)
